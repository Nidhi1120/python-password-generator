# Password Generator using Python

This project is a simple Python-based password generator that creates strong and secure passwords using random characters.

## Features
- Generates random passwords
- Allows user to choose password length
- Uses letters, numbers, and special characters
- Helps create secure passwords for better protection

## Technologies Used
- Python
- random module
- string module

## How it Works
- The user enters the desired password length
- The program generates a random password using a combination of:
  - Uppercase and lowercase letters
  - Digits
  - Special characters
- The generated password is displayed instantly

## Sample Output
Enter password length: 8  
Generated Password: A@7kL!2p
